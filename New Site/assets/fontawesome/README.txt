Place Font Awesome 6.5.1 CSS and webfonts here.

Download from https://fontawesome.com/download and extract:
- all.min.css
- webfonts/ (folder)

Resulting structure:
assets/fontawesome/all.min.css
assets/fontawesome/webfonts/

Update your HTML to use:
<link rel="stylesheet" href="assets/fontawesome/all.min.css">
